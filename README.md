# SYMFONY 4 test task
1. Install project:
```sh
$ git clone https://github.com/alexsobolenko/symfony4-test.git your_project
$ cd your_project
```
2. Configure `DATABASE_URL` parameter in `.env` file.
3. Create database and run server:
* First run:
```sh
make first-run
```
* Another:
```sh
make run
```
